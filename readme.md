## Requirements

- Nodejs
- Markdown files
- Mp3 or wav public urls

## Demo

Install the tool

```
npm install -g github:/podcastjs/podcastjs
```

Create new site

```
podcastjs --new-site acme
cd ./acme
```

Start as developer

```
podcastjs --start
```

By default, it creates a **site** folder. If you need a custom folder:

```
podcastjs --publish docs
```
